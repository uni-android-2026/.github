# Разработка Android-приложений — лабораторные работы

## Описание
Эта организация содержит набор лабораторных работ по дисциплине "Разработка Android-приложений".
В рамках предмета мы последовательно осваиваем базовые элементы Android-разработки.

## Структура
Именование репозиториев в организации отражает номер лабораторной работы, порядок выполнения и название проекта. Используется единый шаблон:

`lab_{номер_лабораторной}.{номер_проекта_внутри_лабораторной}-{название_проекта}`

Пример: `lab_1.1-HelloWorld` означает первую лабораторную работу, первый проект в её составе, с названием **HelloWorld**.
